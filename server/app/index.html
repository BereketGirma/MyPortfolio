AutoShift

AutoShift is a cross platform desktop application designed to automate timesheet entries for Minnesota State Univeristy, Mankato. Built with TypeScript, Electron, and React using Vite framework, it streamlines the process of logging work hours by automating input into the university's timesheet website. AutoShift utilizes Selenium with ChromeDriver for automation, ensuring accuracy and efficieny while reducing manual effort. The app supports Windows and macOS, and operates after the user manually signs in, making it a seamless tool for improving workflow and productivity.
